## 📌 Overview

This project demonstrates large-scale data processing using PySpark by analyzing a synthetic dataset containing 1 million sales records. The project focuses on extracting meaningful business insights while showcasing the scalability and efficiency of distributed computing.

---

## 🎯 Objective

To perform big data analysis on a large dataset and demonstrate how PySpark efficiently handles large-scale data processing.

---

## 🧩 Key Areas Covered

• Large dataset generation (1 million records)
• Data processing using PySpark
• Aggregation and transformation of data
• Sales and customer behavior analysis
• Scalability demonstration using distributed computing

---

## 📊 Analysis Performed

• Total number of records
• Total sales per product
• Average order value per country
• Most popular product

---

## 🔍 Key Insights

• Laptop and Mobile products generate the highest revenue
• Average order value varies across countries
• Certain products dominate customer demand
• PySpark efficiently processes large datasets
